# キーバインド一覧

`<leader>` キーは **スペース** に設定されています。

## Neovim

### 基本操作

| キー | 動作 |
|------|------|
| `<leader>e` | ファイルエクスプローラー (Neo-tree) のトグル |
| `gcc` | 現在行のコメントトグル |
| `gc` (ビジュアルモード) | 選択範囲のコメントトグル |
| `<C-c>` (ビジュアルモード) | クリップボードにコピー |

### ファイル・テキスト検索 (Telescope)

| キー | 動作 |
|------|------|
| `<leader>ff` | ファイル名で検索 |
| `<leader>fg` | ファイル内容をテキスト検索 (live grep) |
| `<leader>fb` | 開いているバッファ一覧 |
| `<leader>fh` | ヘルプタグ検索 |
| `<leader>fs` | ドキュメントシンボル検索 |

Telescope ウィンドウ内:

| キー | 動作 |
|------|------|
| `<C-j>` | 次の候補へ |
| `<C-k>` | 前の候補へ |
| `<CR>` | 選択 |
| `<Esc>` | 閉じる |

### LSP (コード操作)

| キー | 動作 |
|------|------|
| `gd` | 定義へジャンプ |
| `gr` | 参照一覧 (Telescope) |
| `K` | ホバー情報（型や説明の表示） |
| `<F2>` | シンボルのリネーム |
| `<F4>` | コードアクション |
| `gl` | 行の診断情報を表示 |
| `[d` / `]d` | 前/次の診断（エラー・警告）へジャンプ |

### フォーマット

| キー | 動作 |
|------|------|
| `<leader>cf` | 手動フォーマット |
| (自動) | ファイル保存時に自動フォーマット |

### Git 操作 (Gitsigns)

| キー | 動作 |
|------|------|
| `]h` / `[h` | 次/前の変更箇所 (hunk) へジャンプ |
| `<leader>gs` | 現在の hunk をステージ |
| `<leader>gr` | 現在の hunk をリセット |
| `<leader>gS` | バッファ全体をステージ |
| `<leader>gp` | hunk のプレビュー |
| `<leader>gb` | 行の blame 表示 |
| `<leader>gd` | Diff ビューを開く (Diffview) |
| `<leader>gh` | 現在ファイルの Git 履歴 (Diffview) |

### Treesitter テキストオブジェクト

| キー | 動作 |
|------|------|
| `af` / `if` | 関数の外側/内側を選択 |
| `ac` / `ic` | クラスの外側/内側を選択 |
| `aa` / `ia` | 引数の外側/内側を選択 |
| `]f` / `[f` | 次/前の関数へジャンプ |
| `]c` / `[c` | 次/前のクラスへジャンプ |
| `<leader>a` / `<leader>A` | 引数を次/前と入れ替え |

### ウィンドウ管理

| キー | 動作 |
|------|------|
| `<leader>sv` | ウィンドウを縦分割 |
| `<leader>sh` | ウィンドウを横分割 |
| `<leader>se` | 分割サイズを均等化 |
| `<leader>sx` | 現在のウィンドウを閉じる |
| `<leader>h/j/k/l` | 左/下/上/右のウィンドウへ移動 |

### タブ操作

| キー | 動作 |
|------|------|
| `<leader>to` | 新しいタブを開く |
| `<leader>tx` | 現在のタブを閉じる |
| `<leader>tn` | 次のタブへ |
| `<leader>tp` | 前のタブへ |
| `<leader>tf` | 現在のバッファを新しいタブで開く |

### 補完 (nvim-cmp)

| キー | 動作 |
|------|------|
| `<C-n>` | 次の補完候補 |
| `<C-p>` | 前の補完候補 |
| `<CR>` | 補完を確定 |
| `<C-e>` | 補完を閉じる |
| `<C-Space>` | 補完を手動起動 |
| `<C-d>` / `<C-f>` | ドキュメントのスクロール |

### インクリメンタル選択 (Treesitter)

| キー | 動作 |
|------|------|
| `<C-Space>` | 選択を開始/拡大 |
| `<BS>` | 選択を縮小 |

## Tmux

プレフィックスキーは **Ctrl+a** です（デフォルトの `Ctrl+b` から変更済み）。

### ペイン操作

| キー | 動作 |
|------|------|
| `Prefix + \|` | ペインを縦分割 |
| `Prefix + -` | ペインを横分割 |
| `Prefix + h/j/k/l` | ペイン間を移動 |
| `Prefix + H/J/K/L` | ペインサイズを変更 |

### ウィンドウ操作

| キー | 動作 |
|------|------|
| `Prefix + c` | 新しいウィンドウを作成 |
| `Prefix + Ctrl-h` | 前のウィンドウへ |
| `Prefix + Ctrl-l` | 次のウィンドウへ |

### コピーモード

| キー | 動作 |
|------|------|
| `Prefix + [` | コピーモードに入る |
| `v` | 選択開始 (vi モード) |
| `y` | コピー |
| `Ctrl-u` / `Ctrl-d` | ページアップ/ダウン |

### その他

| キー | 動作 |
|------|------|
| `Prefix + r` | 設定ファイルを再読み込み |

## Zsh

| キー/コマンド | 動作 |
|-------------|------|
| `Ctrl+R` | コマンド履歴検索 (Atuin) |
| `g` | ghq + fzf でリポジトリ移動 |
| `vim` | nvim のエイリアス |
| `sshf` | fzf で SSH ホストを選択して接続 |
